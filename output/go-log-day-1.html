Type declaration

@2:40 PM

type declaration is slightly different in go than other C family lanaguages. just in first glance what I got is in C

int x,  int main(int arg, char argc[])

in go

x int,  func main(arg int, argv []char) int

The tutorial linked a nice article discussing the difference of type declaration in go from C. Go's Declaration Syntax

var

@3:26 PM

var is used to declare a list of variables. I've seen this in C# and javascript. var i,j int There is an interesting way to initialize the variable. var c, python, java = true, false, "no!"

there is := short assignment statement. k := 3 without explicit type declaration. But it can only be used inside the function body while var is allowded inside package and function.
